📊 Server Information

Host: 127.0.0.1:8080

Features:

  • ✅ Multi-threaded with configurable thread pool
  • ✅ GET requests for HTML, PNG, JPEG, TXT files
  • ✅ POST requests for JSON file uploads
  • ✅ Path traversal protection
  • ✅ Host header validation
  • ✅ Keep-alive connections with 30s timeout
  • ✅ Request limit (100 per connection)
  • ✅ Comprehensive logging

🧪 Testing Commands

Start Server:

python http_server.py --host 127.0.0.1 --port 8080 --max-threads 10

Test GET Request:

curl -H "Host: 127.0.0.1:8080" http://127.0.0.1:8080/test.png

Test POST Request:

curl -X POST -H "Content-Type: application/json" -H "Host: 127.0.0.1:8080" -d '{"test": "data"}' http://127.0.0.1:8080/
